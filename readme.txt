This plugin creates a multi-instance widget for inserting pod data into your sidebar. Requires the PodsCMS plugin. 

== Description ==

Allows you to insert data from any PodCMS pod into your sidebar. Simply select the pod, the template, and specify the number you would like to show. You can even use "Find Records" to filter the listing.

== Installation ==

Upload folder to your plugins directory.
